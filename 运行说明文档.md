# 系统监控Web面板 - 运行说明文档

## 项目概述

系统监控Web面板是一个基于React和ECharts的实时系统监控应用，提供服务器性能监控、任务管理、告警展示和数据可视化等功能。

## 技术栈

- **前端框架**: React 19.2.0
- **构建工具**: Vite 7.2.4
- **状态管理**: Zustand 5.0.9
- **图表库**: ECharts 6.0.0 + echarts-for-react 3.0.5
- **UI图标**: Lucide React 0.562.0
- **开发语言**: TypeScript

## 环境要求

- **Node.js**: 版本 18.0 或更高
- **npm**: 版本 8.0 或更高（或使用 yarn/pnpm）
- **浏览器**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+

## 安装与运行

### 1. 克隆项目

## 功能使用说明

### 1. 实时监控

- **服务器状态**: 在左侧边栏查看服务器列表和实时状态
- **系统状态**: 在顶部区域查看系统整体健康状态和关键指标
- **告警信息**: 在左侧边栏查看最新告警，按时间倒序排列

### 2. 数据过滤

- **搜索功能**: 在顶部控制台使用搜索框实时过滤服务器、任务和告警
- **支持搜索字段**: 服务器名称、ID、区域、标签等

### 3. 数据流控制

- **暂停/启动**: 点击顶部控制台的暂停/启动按钮控制数据更新
- **刷新**: 点击刷新按钮立即更新所有数据

### 4. 任务管理

- **任务列表**: 在左侧边栏查看任务列表，支持滚动浏览
- **任务详情**: 点击任务项查看详细信息，包括进度、时间、错误信息等

### 5. 数据可视化

- **图表类型切换**: 支持折线图、柱状图、饼图、仪表盘
- **视图维度切换**: 支持按服务器、区域、服务维度查看数据
- **指标维度切换**: 支持查看CPU、内存、磁盘、网络指标
- **历史回放**: 
  - 点击"历史模式"启用历史数据查看
  - 使用播放/暂停/重置按钮控制历史回放
  - 调节回放速度（0.5x、1x、2x、4x）
  - 使用进度条跳转到特定时间点

## 常见问题

### 1. 端口冲突

如果5173端口被占用，Vite会自动使用其他端口。请查看控制台输出的实际访问地址。

### 2. 数据不更新

- 检查数据流是否处于启动状态（顶部控制台显示"启动"按钮表示已暂停）
- 尝试点击刷新按钮手动更新数据
- 检查浏览器控制台是否有错误信息

### 3. 图表不显示

- 确保浏览器支持WebGL（ECharts依赖）
- 检查浏览器控制台是否有JavaScript错误
- 尝试刷新页面

### 4. 性能问题

- 如果页面卡顿，尝试关闭历史模式或减少显示的数据量
- 在生产环境中使用构建版本而非开发版本

## 开发指南

### 1. 添加新组件

在 `src/components` 目录下创建新组件文件夹，包含 `.tsx` 和 `.module.css` 文件。

### 2. 状态管理

使用 Zustand 进行状态管理，在 `src/stores` 目录下创建新store。

### 3. 样式规范

使用 CSS Modules 进行组件样式管理，遵循现有的命名约定。

### 4. 类型定义

在 `src/types` 目录下添加 TypeScript 类型定义。
